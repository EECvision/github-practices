Learning Git hub best practices
  • git init: initializes a repo
  • git config: configures a name to be used with your commits
  • git clone: used to get a copy of a repo
  • git add: used to add changes to the staging area
  • git status: used to track changes based on the file names
  • git diff: used to track changes based on the contents of a file. Also changes which are not yet staged
  • git diff firstbranch secondbranch: used to track changes in the two files after commit
  • git commit
